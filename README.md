### **Prerequisites**
- [Ollama](https://ollama.ai/download) - AI model serving
- Node.js 18+ - Frontend framework
- Python 3.8+ - Backend framework

### **Step 1: Install Ollama**

```bash
cd backend

python server_enhanced.py
```

### **Step 3: Frontend Setup**
